# Best Friends Forever App 🌟

A beautiful scrapbook and tribute template built with React, Vite, and Tailwind CSS.
Features multiple aesthetic layouts and animations to celebrate a special friendship.

## Features Let's Go
- 🌟 **Interactive animations** with Framer Motion.
- 📱 **Fully responsive design** covering Mobile & Desktop.
- 🖼️ **Multiple distinct scrapbook styles:**
  - Standard Minimal layout
  - Premium Green Border style
  - Dark Neon Crowns aesthetic
- 🚀 **PWA Ready:** Installable on devices with a custom manifest and service worker.
- 🎵 **Fake Music Player UI** built in.

## Deployment
Includes a `vercel.json` for immediate Single Page App deployment to Vercel.

## Setup Instructions

```bash
npm install
npm run dev
```
